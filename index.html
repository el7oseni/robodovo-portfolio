Project Overview

RoboDovo is an autonomous delivery robot designed to navigate university campuses and deliver items. The robot processes all tasks using a Lenovo ThinkCentre Mini PC, which handles navigation, object recognition, and more.

Techniques and Technologies

  • Hardware: ThinkCentre Mini PC, LiDAR, Camera, Ultrasonic sensors
  • Software: ROS, YOLO for object detection, SLAM for mapping
  • Algorithms: TEB Planner, Dynamic Window Approach for collision avoidance
  • Power: 12V battery system
